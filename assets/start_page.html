Helidon project generator for Visual Studio Code

This extension helps users to generate their first Helidon project.

Requirements

  • Maven 3.5.3+ is required to be installed on your system.
  • The Java Extension Pack optional but recommended.
  • Helidon CLI is required if you want to use continuous compilation and application restart (applicable for Linux and MacOS).

How to start

  • Open Command Palette by shortcut you are using (default: Ctrl+Shift+P)
  • Type in ‘Helidon: Generate a Helidon project’
    • Follow generator till the end
    • Your new project is generated
  • Type in ‘Helidon: Start page’ to open start page
  • Type in ‘Helidon: Start helidon dev’ to start dev server and to use continuous compilation and application restart
